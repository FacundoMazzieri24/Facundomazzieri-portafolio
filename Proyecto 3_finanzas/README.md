# Proyecto 3 — Análisis Financiero Personal (Excel + Power BI)

## Descripción del Proyecto
Este proyecto consiste en un análisis completo de ingresos, gastos y comportamiento financiero usando Excel para la preparación del dataset y Power BI para la visualización.

El objetivo principal es entender:
- Cómo varían los ingresos y gastos mes a mes.
- Qué categorías representan mayor gasto.
- La evolución del profit total.
- El rendimiento de productos y segmentos.
- Tendencias financieras relevantes para la toma de decisiones.

El análisis incluye dashboards dinámicos que permiten filtrar por fechas, segmento y categorías.

---

## Objetivos del Análisis
- Analizar ingresos, gastos y profit por mes y año.
- Identificar categorías más costosas o más rentables.
- Evaluar el comportamiento financiero por región o país.
- Crear un dashboard claro y visualmente profesional para entender tendencias.
- Detectar oportunidades de ahorro o mejora.

---

##  Herramientas Utilizadas
- **Excel**
  - Limpieza del dataset.
  - Organización de columnas.
  - Estructura para importación en Power BI.
- **Power BI**
  - Visualizaciones avanzadas.
  - Medidas y KPIs.
  - Segmentadores por fecha, país, segmento y producto.
  - Mapas y gráficos comparativos.

---

##  Visualizaciones del Dashboard
El dashboard incluye:
- Línea temporal del profit por mes.
- Mapa global del profit por país.
- Barras de ventas por producto y segmento.
- Segmentadores de fecha.
- Gráficos comparativos por categoría y región.

---

##  Principales Insights
- Países fuertes como Canadá, Alemania y China lideran el profit.
- Productos como Paseo, VTT y Carretera generan los mayores ingresos.
- El promedio mensual muestra fluctuaciones estacionales útiles para planificación.
- El profit crece entre 2013 y 2014 con tendencias claras de expansión.
